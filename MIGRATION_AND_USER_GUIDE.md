# 项目档案本地管理系统：迁移与使用指南

## 1. 当前版本能否使用

当前版本已可用于初步正式测试，支持：

- 项目资料新增、编辑和删除；
- 项目下档案台账新增、编辑、删除；
- 一条档案绑定一个本地电子文件，只保存绝对路径，不复制原文件；
- 打开电子文件、打开所在文件夹、路径失效后重新绑定；
- 整条档案借阅与归还，并保留借阅历史；
- 按项目名、盒号、文件名、当前借用人全局搜索；
- Excel 批量导入档案及按项目导出台账。

测试阶段建议先使用少量真实项目副本，不要立即替代唯一的正式台账。

## 2. 迁移前准备

### 2.1 整理电子文件

建议先建立稳定的文件目录，例如：

```text
D:\项目档案\
├── 城北综合体\
│   ├── A-001 建设工程施工合同.pdf
│   └── A-002 监理规划.docx
└── 南区道路工程\
    └── B-001 竣工验收报告.pdf
```

导入或绑定后不要随意移动、改名或删除这些文件。程序保存的是本机绝对路径；
如果路径变化，档案记录仍会保留，但打开时会提示重新绑定。

### 2.2 备份原始资料

迁移前请保留：

- 原 Excel 台账副本；
- 原电子文件目录副本；
- 原借阅记录。

系统不会自动删除或复制原电子文件。

## 3. 迁移已有项目

当前版本按“一个项目一次导入”处理。

### 第一步：建立项目

1. 启动系统，点击顶部“新增项目”。
2. 填写项目名称，以及建设单位、合同金额、监理单位、开工日期等资料。
3. 保存后，在左侧选中该项目。

### 第二步：准备 Excel

使用随安装包提供的 `档案迁移导入模板.xlsx`。其中“档案导入”工作表的列名必须保持不变：

| 列名 | 是否必填 | 示例 |
|---|---:|---|
| 盒号 | 否 | A-001 |
| 文件名称 | 是 | 建设工程施工合同 |
| 载体类型 | 否 | 原件 |
| 份数描述 | 否 | 3份、2本 |
| 电子文件路径 | 否 | D:\项目档案\城北综合体\合同.pdf |

注意：

- 每行代表一条完整档案，不要把一条档案拆成多个数量行；
- “份数描述”允许文字；
- 电子文件路径必须是将来运行程序的 Windows 电脑上的绝对路径；
- 没有电子文件时可留空，之后在系统中逐条绑定；
- 不要增加标题行、合并单元格或修改表头。

### 第三步：导入

1. 在左侧选择目标项目。
2. 点击档案表格上方的“Excel 导入”。
3. 选择填写完成的模板。
4. 查看成功和失败条数；失败行会显示原因。
5. 抽查盒号、文件名称、份数和电子文件路径。

重复导入同一个 Excel 会产生重复档案。正式导入前建议先用 3–5 行测试。

### 第四步：迁移借阅状态

当前 Excel 导入只迁移档案台账，不自动迁移历史借阅记录。

对于当前仍在借出的档案：

1. 找到对应档案；
2. 点击“借阅”；
3. 填写实际借用人、联系电话、事由和原借出日期。

已归还的历史借阅记录暂不支持批量导入。

## 4. 日常使用

### 项目管理

- 左侧列表切换项目；
- 右上区域可直接编辑 9 个项目字段；
- 编辑后点击“保存项目资料”；
- 删除项目会同时删除其档案和借阅记录，请谨慎操作。

### 档案管理

- “新增档案”：录入盒号、名称、载体类型、份数和电子文件；
- “编辑档案”：修改当前选中档案；
- “绑定/重新绑定”：选择本机现有文件，系统不复制文件；
- “打开”：使用 Windows 默认程序打开电子文件；
- “所在文件夹”：在资源管理器中定位文件；
- “删除档案”：同时删除该档案的借阅历史，不删除磁盘上的电子文件。

### 借阅与归还

- 在库档案显示“🟢 在库”；
- 借出档案显示“🔴 已借出：借用人”；
- 一条档案只能存在一条未归还借阅记录；
- 系统不支持部分数量借出；
- 点击“归还”并填写归还日期后，状态自动恢复为在库。

### 全局搜索

顶部搜索框支持项目名、盒号、文件名和当前借用人。停止输入约 300ms 后显示跨项目结果；
单击结果行会自动切换到对应项目并选中档案。

### Excel 导出

选择项目后点击“Excel 导出”，可导出该项目当前台账及动态借阅状态。

## 5. 数据备份与换电脑

Windows 数据库默认位于当前用户目录：

```text
%LOCALAPPDATA%\ProjectArchiveManager\data\project_archives.db
```

备份时：

1. 先完全退出程序；
2. 复制整个 `data` 文件夹；
3. 同时备份电子文件目录。

换电脑后，即使盘符或目录发生变化，项目和档案记录仍可恢复，但原绝对路径可能失效，
需要逐条重新绑定。建议不同电脑保持相同盘符与目录结构。

## 6. 初步测试建议

建议按以下顺序验收：

1. 新建一个测试项目并保存 9 个项目字段；
2. 手工新增两条档案；
3. 绑定 PDF、Word 或 Excel 文件并测试打开；
4. 借出一条档案，确认不能重复借阅；
5. 搜索借用人并测试跨项目跳转；
6. 归还档案，确认状态恢复；
7. 使用模板导入 3–5 条档案；
8. 导出 Excel 并核对内容；
9. 备份数据库后再进行较大批量迁移。

## 7. 当前版本限制

- 仅支持 Windows 单机本地使用，不支持多人共享数据库；
- 不自动复制、上传或同步电子文件；
- 不支持一条档案绑定多个主文件；
- 不支持部分数量借出；
- 暂不支持批量导入历史借阅记录；
- 安装包如未进行代码签名，Windows SmartScreen 可能显示未知发布者提示。
